# Visual & Editing Retention Patterns

A script provides the audio, but visuals retain the eye.

## 1. The "Satisfying Loop"
*   **Technique:** The last visual frame matches the first visual frame.
*   **Usage:** Best for "Deep Dive" scripts.
*   **Example:** If the video starts with pouring coffee, end the video with the coffee cup being set down, ready to pour again.

## 2. Text-Overlay Hierarchy
*   **The Headline:** 0:00-0:03. Must match the spoken hook but be shorter. (Audio: "Stop doing cable flys like this" -> Text: "STOP DOING THIS").
*   **The Keywords:** Highlight only the noun/verb in the sentence. Don't caption every word unless it's a "talking head" style.
*   **The Safe Zone:** Keep text centered, avoiding the UI overlay (Like/Comment buttons) on the right and bottom.

## 3. The "B-Roll" Ratio
*   **Talking Head:** Max 3 seconds before a cut.
*   **Visual Proof:** When explaining a "Mechanism," you MUST show it. Don't talk about a "spreadsheet error" while looking at a face. Show the spreadsheet.
*   **Green Screen:** Use for "Reacting" to news articles or comments. High credibility signal.

## 4. Editing Pacing
*   **J-Cuts:** Audio from the next scene starts 0.5s before the visual cuts. (Smoother flow).
*   **Zoom Cuts:** Subtle digital punch-in (1.1x zoom) on emphatic words to simulate energy.
*   **The "Millennial Pause" Ban:** Audio must start on Frame 1. No breathing in before speaking.
